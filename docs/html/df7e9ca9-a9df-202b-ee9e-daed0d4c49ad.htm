Parameters

numberOfColumnsFromLeft
Type: SystemNullableInt32
Number of columns from left of the worksheet to the split line. The particular column widths are considered.
The parameter is nullable. If left null, the method acts identical to SetHorizontalSplit(Int32, Boolean, Address, WorksheetWorksheetPane)
numberOfRowsFromTop
Type: SystemNullableInt32
Number of rows from top of the worksheet to the split line. The particular row heights are considered.
The parameter is nullable. If left null, the method acts identical to SetVerticalSplit(Int32, Boolean, Address, WorksheetWorksheetPane)
freeze
Type: SystemBoolean
If true, all panes are frozen, otherwise remains movable
